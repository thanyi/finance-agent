# analyze-stock

对指定A股标的执行完整的M1-M6价值投资深度分析。

## 使用方式

```
/analyze-stock [股票代码]
示例：/analyze-stock 002415（海康威视）
```

---

## 执行流程（SOP）

```
Step 1  调用 /stock-data-collect [股票代码]，完成数据采集
         ↓
Step 2  数据质量核查（异常值标注⚠️，缺失项标注[数据缺失：原因]）
         ↓
Step 3  M1-M6 六模块分析（严格按各模块框架执行）
         ↓
Step 4  输出各模块结论（含评级⭐，含数据来源）
         ↓
Step 5  汇总综合结论（按本文输出模板）
         ↓
Step 6  数据纪律自查（无编造，无估算，无缺失未标注）

⚠️ 每一步若数据无法获取，必须停下来标注，不得跳过。
```

## M1｜经济护城河分析（20%）

> 哲学来源：巴菲特"护城河"概念——决定企业能否长期维持超额利润。

**1.1 护城河类型识别**（逐一验证，必须用数据支撑：有 / 弱 / 无）

| 护城河类型 | 验证指标 | 数据要求 |
|---|---|---|
| 品牌溢价 | 毛利率高于行业均值 | 毛利率数据 |
| 网络效应 | 用户增长与价值正反馈 | 用户数据（若有披露）|
| 成本优势 | 单位成本低于同行 | 同行对比数据 |
| 转换成本 | 客户流失率、续约率 | 年报披露 |
| 规模效应 | 市占率 & 固定成本摊薄 | 行业市占率 |
| 无形资产 | 专利数、特许经营权 | 年报披露 |
| 政策壁垒 | 牌照稀缺性 | 监管文件 |

**1.2 护城河宽度评估**
- 近5年ROE（扣非）稳定性：稳定高于15%是强护城河信号
- 毛利率趋势：扩张 / 收窄 / 承压
- 市场份额变化：扩大 / 稳定 / 流失

**1.3 护城河威胁**
- 技术颠覆风险
- 政策收紧风险（A股特有：行业监管骤变）
- 外资/跨界竞争者进入

## M2｜财务健康度 & 盈利质量分析（25%）

> 哲学来源：格雷厄姆《证券分析》严格财务标准 + A股财务造假识别体系

**2.1 核心财务指标（近5年趋势，禁止缺失）**

盈利能力：

| 指标 | 计算方式 | 格雷厄姆标准 |
|---|---|---|
| 扣非ROE | 扣非净利润 / 平均净资产 | >15% 优秀，连续5年>12%为强质地 |
| 毛利率 | 毛利润 / 营收 | 行业对比，重点看趋势 |
| 净利率 | 扣非净利润 / 营收 | >10% 优选 |
| ROA | 净利润 / 平均总资产 | >6% 健康 |

偿债能力：

| 指标 | 计算方式 | 警戒线 |
|---|---|---|
| 资产负债率 | 总负债 / 总资产 | >70% 高风险（制造业）；金融业单独评估 |
| 有息负债率 | 有息负债 / 总资产 | >40% 警惕 |
| 流动比率 | 流动资产 / 流动负债 | <1.0 危险 |
| 利息保障倍数 | EBIT / 利息支出 | <3倍 警惕 |

现金流质量：

| 指标 | 说明 | 健康标准 |
|---|---|---|
| FCF（自由现金流）| 经营CFO - 资本支出 | 连续5年为正 |
| 净利润现金含量 | 经营CFO / 净利润 | >80% 健康，<60% 警惕 |
| 应收账款增速 vs 营收增速 | 应收增速不应持续高于营收 | 若高于则⚠️ |

运营效率：应收账款周转天数 / 存货周转天数 / 总资产周转率（均需对比行业均值）

**2.2 A股财务造假识别**（必查，出现2项以上须重点警示）

```
□ 营收高增长但经营现金流持续为负或远低于净利润
□ 应收账款增速连续3年高于营收增速30%以上
□ 存货大幅增加但营收未同步增长
□ 在建工程长期挂账（超过行业建设周期）
□ 货币资金大量但同时大量有息借款（矛盾资金结构）
□ 关联方应收款占总应收比例超过30%
□ 审计意见非标（须⚠️⚠️标注）
□ 频繁更换审计机构（3年内更换2次以上）
□ 商誉占净资产比例超过50%（并购泡沫风险）
□ 毛利率显著高于同行但无合理解释
□ 净利润连续多年呈现"整数"或过于平滑
□ 控股股东股权质押比例超过60%
□ 实控人存在非经营性占款记录
□ 子公司众多且分布于注册地隐蔽地区（如BVI离岸公司）
```

Beneish M-Score 简化验证（有数据时执行）：

```
DSRI = (应收/营收)_t / (应收/营收)_t-1
GMI  = 毛利率_t-1 / 毛利率_t
AQI  = (1-(流动资产+固定资产)/总资产)_t / (1-(流动资产+固定资产)/总资产)_t-1
SGI  = 营收_t / 营收_t-1
DEPI = 折旧率_t-1 / 折旧率_t
M = -4.84 + 0.92*DSRI + 0.528*GMI + 0.404*AQI + 0.892*SGI + 0.115*DEPI
M > -1.78 → 存在盈余操纵风险 ⚠️
```

**2.3 扣非ROE vs 行业均值**：获取同行业前5公司扣非ROE，计算算术均值，判断目标公司相对位置

```
【M2 财务健康度】评级：⭐⭐⭐
财务造假风险：低/中/高（触发 X 项红旗信号）
⚠️ 异常数据：[列举]
扣非ROE均值（5年）：X%，行业均值：X%，结论：[高于/低于/持平]
FCF状况：连续5年正/部分年度负/持续为负
现金流质量：健康/警惕/异常
数据来源：[列举接口]
```

## M3｜商业模式 & 管理层分析（15%）

> 哲学来源：费雪《怎样选择成长股》"15要点"中对管理层的重视

**3.1 商业模式**
- 收入结构：主营业务清晰度，单一来源风险 vs 多元化
- 定价权：能否主动提价？客户对提价的接受度（费雪核心看点）
- 客户集中度：前5大客户占营收比例（>50%需⚠️）
- 供应商集中度：前5大供应商占采购比例（>70%需⚠️）
- 商业模式可复制性：竞争者进入门槛

**3.2 管理层量化指标**

| 维度 | 数据来源 | 说明 |
|---|---|---|
| 管理层薪酬/净利润比率 | 年报 | 过高需⚠️ |
| 实控人/大股东增减持记录 | 交易所公告 | 大幅减持需⚠️ |
| 股权质押比例 | 巨潮资讯 | >60%需⚠️⚠️ |
| 高管离职频率 | 公告 | 核心管理层频繁离职需⚠️ |
| 股权激励计划 | 公告 | 考核指标是否合理 |
| 分红稳定性 | 分红历史 | 连续10年分红是正信号 |

**3.3 定性评估**（基于公开信息）
- 战略一致性：近5年战略是否清晰、未频繁转向
- 资本配置能力：ROE vs 同期再投资回报率
- 股东回报意识：股息率、回购历史
- 历史承诺兑现率：对比过往业绩指引与实际结果

## M4｜增长天花板 & 驱动力分析（15%）

> 哲学来源：费雪成长股投资理论——寻找长期成长的驱动力

**4.1 行业空间分析**
- 行业生命周期：导入期 / 成长期 / 成熟期 / 衰退期（必须注明依据）
- 行业规模与增速：当前市场规模（必须引用官方/权威来源）
- 渗透率空间：目标公司当前渗透率 vs 可能上限
- 政策催化剂：近2年重大行业政策（来源：国务院、发改委、监管部门）

**4.2 成长驱动力**（无数据支撑的不得写入）

```
□ 市场份额扩张（数据：市占率变化）
□ 产品/服务提价能力（数据：ASP历史变化）
□ 新产品线渗透（数据：各产品线营收贡献）
□ 地理扩张（数据：区域营收变化）
□ 并购整合（数据：并购标的ROI）
```

**4.3 天花板评估**
- 当前营收 / 行业总规模 = 市占率
- 市占率已超40%，增长空间显著收窄，需标注
- 海外扩张可能性（若有，需评估汇率风险和地缘政治风险）

**4.4 A股特有政策风险**

```
□ 是否处于政策强监管行业（教育/游戏/医疗/地产/金融）
□ 是否依赖政府补贴（补贴占利润比例 > 20%需⚠️）
□ 是否存在"运动式"政策风险（如双碳、反垄断等）
□ 是否存在地方政府关系高度绑定风险
```

## M5｜市场情绪 & 筹码结构分析（10%）

> A股特性：散户占比高（个人投资者贡献约80%交易量），情绪波动剧烈，与基本面脱钩频繁。
> **情绪是辅助工具，不是决策依据。**

**5.1 筹码结构评估维度**

| 维度 | 含义 | 判断标准 |
|---|---|---|
| 股东人数变化趋势 | 筹码集散 | 连续3季度↓且股价未涨→谨慎 |
| 机构持仓比例 | 聪明钱认可度 | <5%或持续下降需⚠️ |
| 融资余额/流通市值 | 杠杆情绪 | >5%说明情绪偏热 |
| 北向资金净流向 | 外资动向参考 | 连续流出需关注 |

**5.2 价格与情绪分析**
- 52周价格区间：当前股价处于低位/中位/高位
- 换手率：>10%单日往往是情绪峰值信号
- A股散户特征：追涨杀跌 / 概念炒作 / 政策市 / 年报季信息差

**5.3 情绪极致谨慎原则**

```
【禁止基于情绪做正面判断的情境】
- 股价30个交易日内涨幅超过50%（无论基本面如何）
- 社交媒体出现大规模"百倍股"讨论
- 行业出现"风口"概念集体炒作
- 公司连续涨停（定性：投机行为，非价值发现）

【可信的反向情绪信号（必须配合基本面使用）】
- 股东人数连续增加但股价横盘/下跌（散户接盘，需更谨慎）
- 机构持续减持（smart money离场信号）
- 融资余额快速上升（杠杆情绪峰值）
```

## M6｜估值 & 安全边际分析（15%）

> 哲学来源：格雷厄姆《聪明的投资者》——安全边际是投资的核心原则

**6.1 多维估值**

DCF（自由现金流折现）：

```
⚠️ DCF对假设极度敏感，必须做敏感性分析，禁止单一DCF结论作为买入依据

输入参数（均需基于历史数据，禁止主观假设过于乐观）：
- FCF历史均值（5年）：取实际数据
- 成长率：以近5年FCF CAGR为参考上限，不得人为上调
- 折现率（WACC）：通常10%-12%（A股风险溢价考虑）
- 永续增长率：不超过3%（GDP增速参考）

输出：悲观 / 中性 / 乐观三种假设下的内在价值
```

相对估值：

| 估值指标 | 当前值 | 历史5年均值 | 行业均值 | 结论 |
|---|---|---|---|---|
| PE（TTM）| [数据] | [数据] | [数据] | 高/合理/低估 |
| PB | [数据] | [数据] | [数据] | 高/合理/低估 |
| PS | [数据] | [数据] | [数据] | 参考 |
| EV/EBITDA | [数据] | [数据] | [数据] | 参考 |
| 股息率 | [数据] | [数据] | [数据] | 吸引/一般 |

⚠️ 禁止仅用PE/PB单一指标得出估值结论，必须多维交叉验证

格雷厄姆公式（适用于稳定成熟型公司）：

```
内在价值 = EPS × (8.5 + 2g)
g = 预期年化增长率，以历史实际增长率保守估算，不得超过15%
格雷厄姆安全边际要求：当前股价 ≤ 内在价值 × 66.7%
```

**6.2 安全边际**

```
安全边际 = (内在价值 - 当前股价) / 内在价值 × 100%

> 30%  → 较充足安全边际（格雷厄姆标准：不低于33%）
15-30% → 安全边际有限，需对质地更有把握
< 15%  → 安全边际不足，不符合价值投资买入标准
< 0%   → 当前价格高于内在价值，高估
```

**6.3 A股估值特殊考量**

```
□ 含大量商誉（并购溢价）→ 扣除商誉后重算PB
□ 政府补贴影响净利润 → 用扣非净利润重算PE
□ 大股东减持意愿 → 折价考量
□ 小盘股流动性折价：酌情调整
□ 警惕"价值陷阱"：必须确认存在催化剂或边际改善才考虑布局
```
