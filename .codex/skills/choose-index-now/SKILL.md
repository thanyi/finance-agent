# choose-index-now

**这个 skill 只回答一件事：**
> 这个指数，当前时点，值不值得配置？

它不是完整的 ETF 分析报告，而是帮你在 5 分钟内做出"值得关注 / 继续等待 / 明确回避"的判断。

---

## 使用方式

```
/choose-index-now [指数名称或代码]

示例：
/choose-index-now 创业板指（399006）
/choose-index-now 纳斯达克100
/choose-index-now 沪深300（000300）
/choose-index-now 中证500
```

输入的是**指数**，不是 ETF 代码。

---

## 执行流程

```
Step 1  采集三项核心数据（估值 / 趋势 / 拥挤度）
         ↓
Step 2  A 估值判断 → 这个指数现在贵不贵？
         ↓
Step 3  B 趋势判断 → 趋势是否支持现在配置？
         ↓
Step 4  C 拥挤度判断 → 现在是不是太热？
         ↓
Step 5  三维综合结论 → 值得选 / 继续等待 / 回避
         ↓
Step 6  按需调用子 skill（见触发条件）
```

> ⚠️ 数据缺失须标注，不得估算或跳过。

## A｜估值——这个指数现在贵不贵？

> 估值是"价值锚"，决定你在历史上的哪个位置买入。

**PE / PB 5年分位**

```
通用参考区间（宽基指数）：

< 20%   → 🟢🟢 历史低估，逆向布局窗口
20~40%  → 🟢   相对低估，值得关注
40~60%  → 🟡   历史均值附近，合理但不便宜
60~80%  → 🟠   偏贵，谨慎
> 80%   → 🔴   历史高估，不是好的买入时点

⚠️ 创业板、纳指等成长型指数 PE 天然偏高，
   须使用下方"指数专项参考"，不可套用宽基区间。
```

**指数专项参考**

```
创业板指：
→ PE 绝对值不是关键，核心看【创业板/沪深300 比值】的5年分位
→ 比值分位 < 30% → 成长相对价值有吸引力
→ 比值分位 > 70% → 成长溢价过高，谨慎

纳斯达克100：
→ 用 Forward PE（12个月预期PE）而非 TTM PE
→ Forward PE 5年分位 < 35% → 合理介入区间
→ 同时必须确认：TIPS 实际利率是否处于低位或下行趋势
```

**股债性价比**

```
盈利收益率 = 1 ÷ PE

盈利收益率 > 国债收益率 × 1.5 → 🟢 股票相对债券有吸引力
盈利收益率 ≈ 国债收益率        → 🟡 性价比一般
盈利收益率 < 国债收益率        → 🔴 股票吸引力弱
```

**估值置信度**

```
PE分位 / PB分位 / 股债比价 三者结论一致？

三者一致  → ★★★ 高置信度
两者一致  → ★★☆ 中置信度，结论保守使用
三者分歧  → ★☆☆ 低置信度，等待信号收敛，不宜重仓
```

```
【A 估值小结】
PE 5年分位：[X%]  PB 5年分位：[X%]
股债性价比：[盈利收益率 X%  vs 国债 X%]
置信度：[★★★ / ★★☆ / ★☆☆]
估值结论：[🟢🟢低估 / 🟢偏低 / 🟡合理 / 🟠偏贵 / 🔴高估]
```

## B｜趋势——趋势是否支持现在配置？

> 低估值不代表立刻就能涨。趋势判断决定你的介入时机是否合适。

**RSRS 择时得分**

```
计算方式：
① 近18日 以最低价为X、最高价为Y 做线性回归，得斜率 β 和 R²
② 维护近600日 β 序列，计算当日 Z-Score
③ RSRS得分 = Z-Score × R²

信号解读：
> 0.7    → 🟢 趋势强势，支撑强于阻力
-0.7~0.7 → 🟡 震荡，趋势不明朗
< -0.7   → 🔴 趋势弱势，阻力强于支撑
```

**均线方向确认**

```
今日 MA20 vs 5日前 MA20：
向上 → 🟢 短期趋势向好
横盘 → 🟡 方向不明
向下 → 🔴 短期趋势仍弱

RSRS > 0.7 且 MA20 向上 → ✅ 双重确认，趋势可信
RSRS < -0.7 且 MA20 向下 → ❌ 双重确认，趋势仍弱
两者方向不一致            → ⚠️ 信号冲突，等待一致
```

```
【B 趋势小结】
RSRS 得分：[X]      → [🟢强势 / 🟡震荡 / 🔴弱势]
MA20 方向：[上/横/下] → [🟢 / 🟡 / 🔴]
趋势结论：[✅双重确认强势 / ⚠️信号冲突 / ❌双重确认弱势]
```

## C｜拥挤度——现在是不是太热太挤？

> 拥挤度衡量"当前市场是不是已经挤满了人"，防止在情绪顶部追入。

**北向资金方向**（若可获取）

```
近10日累计净流入 > 0 → 外资持续流入，市场偏暖
近10日累计净流入 < 0 → 外资持续流出，需谨慎
单日净流入 > 50亿     → 明显增量资金入场
```

**换手率观察**

```
近5日换手率 vs 近90日均值：

> 200% 均值 → ⚠️ 短期情绪过热，追涨风险高
100~200%    → 🟡 活跃，正常偏热
50~100%     → 🟢 温和，正常区间
< 50%       → 🟢🟢 低迷，市场冷清（逆向关注信号）
```

**拥挤度快速判断**

```
近30日指数涨幅 > 20% 且换手率 > 均值150%
→ ⚠️ 短期情绪可能已透支，即便估值合理也须降低仓位

估值低位 且 换手率低迷 且 北向持续流出
→ 🟢 悲观情绪可能过度，逆向布局信号
```

> ⚠️ 若北向资金数据无法获取，仅用换手率判断，并标注数据缺失。

```
【C 拥挤度小结】
北向资金近10日：[净流入X亿 / 净流出X亿 / 数据缺失]
换手率 vs 90日均：[X%]  → [🟢低迷 / 🟡正常 / ⚠️过热]
近30日涨幅：[X%]
拥挤度结论：[🟢市场冷清 / 🟡正常 / ⚠️偏热 / 🔴过热追高风险]
```

## 子 skill 调用条件

完成以上三维判断后，按需调用：

```
调用 check-entry-timing 的情况：
→ 综合结论为 🟢🟢 或 🟢，想要更精确的介入时点
→ 想知道创业板/纳指的专项择时矩阵触发了几项
→ 想加入动量因子做更完整的择时确认

调用 check-etf-quality 的情况：
→ 决定配置该指数，需要在同指数多个 ETF 中选工具
→ 当前 ETF 溢价率异常或规模较小，需要核查
→ 没有现成持仓，第一次为该指数选 ETF
```
