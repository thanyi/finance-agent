## ⚠️ 铁律：数据纪律（最高优先级）

```
【绝对禁止】
- 禁止编造任何财务数据、行业数据、预测数据
- 禁止在未找到数据时"估算""推测""根据常识补充"
- 禁止使用"大约""可能约为""预计约"等模糊表述替代真实数据
- 禁止将自然语言描述性信息转化为伪精确数字

【必须执行】
- 所有数据必须标注来源（平台名称 + 接口/页面路径 + 查询时间）
- 所有异常数据必须用 ⚠️ 标注并说明异常原因
- 若某项数据无法获取，必须明确写出"[数据缺失：原因]"，不得跳过
- 事实陈述使用数据，观点输出必须附完整逻辑链
- 所有预测必须基于行业规律 + 历史均值，并标注置信区间

【数据优先级】
1. 官方披露（交易所公告、招股书、年报原文 PDF）
2. 监管数据库（巨潮资讯、上交所/深交所官网）
3. 主流金融平台（东方财富、同花顺、雪球）
4. 第三方 API（AKShare、BaoStock、Tushare 免费层）
5. 实在无法获取 → 写明缺失，不推测
```

---

## 使用方式

分析A股标的，执行完整M1-M6价值投资框架：

```
/analyze-stock [股票代码]
示例：/analyze-stock 002415
```

该 skill 包含完整分析框架，会自动调用 `/stock-data-collect` 进行数据采集。
