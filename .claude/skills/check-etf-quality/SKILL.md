# check-etf-quality

**这个 skill 只回答一件事：**
> 我决定配置这个指数了，哪个 ETF 更适合作为实现工具？

它聚焦于 ETF 作为"工具"的质量，不重复做指数层面的估值和择时分析。

---

## 使用方式

```
/check-etf-quality [指数名称 或 多个ETF代码]

示例一：给我找追踪创业板的ETF并比较
/check-etf-quality 创业板指（399006）

示例二：指定几个ETF横向比较
/check-etf-quality 159915 vs 159952 vs 159967

示例三：只查一个ETF是否够格
/check-etf-quality 513100（纳指ETF）
```

---

## 何时调用这个 skill？

```
✅ 需要调用的情况：
→ 决定配置某指数，但不知道选哪个 ETF（同指数有多个可选）
→ 持有某 ETF 但规模缩水、溢价异常，想确认是否需要换仓
→ 第一次为某个指数配置，没有现成持仓

⏩ 不需要调用的情况：
→ 已持有该指数的主流大型 ETF（规模>50亿），近期无异常
→ 只是想判断"该指数现在该不该买"（用 choose-index-now）
→ 只是想确认介入时点（用 check-entry-timing）
```

```
对每个候选 ETF 采集：
```

## 核查维度一：这个 ETF 安全吗？

**规模判断**

```
AUM > 50亿  → ✅ 健康，流动性充分有保障
AUM 10~50亿 → ✅ 正常运营
AUM 2~10亿  → 🟡 偏小，关注规模变化趋势
AUM < 2亿   → ⚠️ 清盘警戒线，不建议配置

⚠️ 清盘触发：连续60个工作日净值低于5000万元
规模缩减须区分原因：净值下跌（被动）vs 持续净赎回（主动流出，更需警惕）
```

**费率判断**

```
总费率 = 管理费率 + 托管费率

宽基 ETF 行业均值：0.15%~0.50%/年
行业/主题 ETF 均值：0.50%~1.00%/年

总费率 ≤ 行业均值       → ✅ 合理，长期持有友好
总费率 > 行业均值×1.5   → ⚠️ 长期成本偏高，同指数若有更低费率选项则优先
```

```
【安全性小结】
AUM：[X亿]  状态：[✅健康 / 🟡偏小 / ⚠️警戒]
总费率：[X%]  vs 行业均值：[低/持平/偏高]
```

## 核查维度二：这个 ETF 跟得准吗？

**跟踪误差（TE）**——日常波动层面的偏离

```
年化 TE = StdDev(ETF日收益率 - 指数日收益率) × √252

< 0.3%    → ⭐⭐⭐⭐⭐ 旗舰水准
0.3~0.8%  → ⭐⭐⭐⭐  良好
0.8~1.5%  → ⭐⭐⭐   可接受
> 1.5%    → ⚠️ 较差，可能存在额外损耗
```

**跟踪偏差（TD）**——长期收益层面的偏离

```
近1年 TD = ETF 年化收益率 - 指数年化收益率
理论合理值 ≈ -(管理费率 + 托管费率)

TD ≈ -费率         → ✅ 正常，如实反映费率损耗
TD 明显优于 -费率   → ✅ 有额外收益（如证券出借、打新收益）
TD 明显差于 -费率   → ⚠️ 存在额外损耗，需查明原因
```

```
【跟踪质量小结】
年化 TE：[X%]  → [⭐⭐⭐⭐⭐ / ⭐⭐⭐⭐ / ⭐⭐⭐ / ⚠️]
近1年 TD：[X%]  vs 费率 -[X%]  → [✅正常 / ✅有超额 / ⚠️有损耗]
复制方式：[完全/抽样/合成]
```

## 核查维度三：这个 ETF 买得进卖得出吗？

```
日均成交额（近3个月）：

> 10亿   → ✅✅ 旗舰级，机构也能大额操作
1~10亿   → ✅  普通投资者无障碍
0.1~1亿  → 🟡  偏低，注意买卖价差
< 0.1亿  → ⚠️  严重不足，慎入

当前溢折价率：
-0.2% ~ +0.2% → ✅ 正常
> +0.5%        → ⚠️ 溢价过高，追高风险
< -0.5%        → 需判断原因（套利机会 or 流动性问题）

申赎状态：
正常开放 → ✅ 套利机制有效，溢折价可自我修复
暂停申购 → ⚠️ 溢价难以消退，谨慎追高
```

```
【流动性小结】
日均成交额：[X亿]  → [✅✅ / ✅ / 🟡 / ⚠️]
溢折价率：[X%]    → [✅正常 / ⚠️偏高 / 需确认]
申赎状态：[正常/暂停申购/暂停赎回]
```

## 横向比较（同指数多 ETF 时）

当同一指数有多个 ETF 可选时，按以下优先级决策：

```
优先级排序逻辑：

第一关（硬性筛选，不达标直接淘汰）：
→ AUM < 2亿 → 排除（清盘风险）
→ 申赎暂停 且 溢价 > 1% → 排除

第二关（主要比较维度）：
→ 总费率：同等条件下，费率低的优先
→ 日均成交额：越大越好（流动性保障）
→ 年化 TE：越小越好（跟踪精准）

第三关（决胜维度，前两关相近时）：
→ TD vs 费率：哪个超额更多（打新/出借收益）
→ 基金公司口碑及运营年限

实用结论：
对于创业板、纳指等主流大型指数，
通常规模最大 + 费率最低 + 成交额最高的那个 ETF
就是最优工具，不需要过度分析。
```
