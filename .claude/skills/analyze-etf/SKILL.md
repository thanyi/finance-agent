# analyze-etf

对指定A股ETF执行完整的M1-M6分析，输出配置价值判断与择时建议。

## 使用方式

```
/analyze-etf [ETF代码]
示例：/analyze-etf 510300（沪深300ETF）
```

---

## 执行流程（SOP）

```
Step 1  调用 /etf-data-collect [ETF代码]，完成数据采集
         ↓
Step 2  确认ETF类型（股票型/债券型/商品型/跨境型）→ 决定适用模块
         ↓
Step 3  M1-M6 六模块分析（按类型跳过不适用模块，注明"[不适用：原因]"）
         ↓
Step 4  输出各模块结论（含评级⭐，含数据来源）
         ↓
Step 5  汇总综合结论（按本文输出模板）
         ↓
Step 6  数据纪律自查（无编造，无估算，无缺失未标注）

⚠️ 每一步若数据无法获取，必须停下来标注，不得跳过。
⚠️ ETF分析与个股分析核心差异：ETF无内在价值，核心判断是"指数便宜吗 + 产品好不好"。
```

## M2｜跟踪质量分析（20%）

> ETF的核心职能是忠实追踪指数，跟踪质量差等同于产品缺陷。

**2.1 跟踪误差（TE）**

```
年化跟踪误差 = StdDev(ETF日收益率 - 指数日收益率) × √252

判断标准（宽基ETF参考，行业ETF可适当放宽）：
< 0.3%  → ⭐⭐⭐⭐⭐ 优秀（旗舰ETF水准）
0.3~0.8% → ⭐⭐⭐⭐  良好
0.8~1.5% → ⭐⭐⭐   一般，可接受
1.5~2.5% → ⭐⭐    较差，产品效率低
> 2.5%  → ⭐     差，跟踪失效 ⚠️
```

**2.2 跟踪偏差（TD）**

```
近1年跟踪偏差 = ETF年化收益率 - 指数年化收益率

理论值 ≈ -(管理费率 + 托管费率)
TD 接近 -费率 → 正常，ETF如实反映费率损耗
TD 显著优于 -费率 → ETF有额外收益来源（证券出借、打新等）
TD 显著差于 -费率 → ⚠️ 额外损耗（流动性拖累、大额申赎冲击）
```

**2.3 复制方式质量**

| 复制方式 | 优点 | 风险点 |
|---------|------|--------|
| 完全复制 | TE最低，透明度高 | 流动性差的成分股拖累 |
| 抽样复制 | 适合成分股多/流动性差的指数 | 抽样误差增加TE |
| 合成复制 | 适合跨境/商品 | 对手方风险（Swap） |

**2.4 分红处理**

```
分红再投资型（积累型）→ 长期持有更有利，追踪更准确
现金分红型 → 分红后需手动再投资，否则存在现金拖累（cash drag）
```

## M3｜流动性 & 溢折价分析（15%）

> 流动性直接影响实际交易成本，溢折价决定是否以公允价格买入。

**3.1 流动性评级**

| 日均成交额 | 评级 | 说明 |
|-----------|------|------|
| > 10亿元  | ⭐⭐⭐⭐⭐ | 旗舰级，机构可大额操作 |
| 5~10亿元  | ⭐⭐⭐⭐  | 良好，普通投资者无障碍 |
| 1~5亿元   | ⭐⭐⭐   | 一般，小额操作正常 |
| 0.1~1亿元 | ⭐⭐    | 偏低，注意买卖价差损耗 |
| < 0.1亿元 | ⭐     | ⚠️ 严重不足，慎入 |

**3.2 溢折价分析**

```
溢价率 = (ETF市场价 - IOPV净值) / IOPV净值 × 100%

IOPV（实时净值）通常由交易所盘中每15秒发布

溢价 > 0.5% → ⚠️ 市价高于净值，追高风险，可等价差收窄再买
折价 < -0.5% → 注意：可能是流动性问题或套利机会，需判断原因
折价 < -1.0% → ⚠️ 需核查是否存在流动性冻结或基金暂停申购
正常范围：-0.2% ~ +0.2%
```

> ⚠️ QDII ETF因海外市场收盘时差常出现持续溢价，须单独评估，不能直接套用上述标准。

**3.3 申赎机制**

```
开放申购赎回 → 套利机制有效，溢折价自我修复
暂停申购 → 场外资金无法买入，溢价难以消退，追高风险高
T+2清算周期 → ETF当日买入，下一交易日可卖出（T+1可卖）
最小申购/赎回单位 → 通常50万元或100万元，主要面向机构套利者
```

## M4｜持仓 & 集中度风险分析（15%）

> ETF的持仓即实际风险暴露，集中度决定单一事件冲击。

**4.1 持仓集中度**

```
前10大持仓合计权重分析：

> 70% → 高度集中，少数股票主导，行业ETF常见
50~70% → 中等集中，行业/主题ETF正常区间
30~50% → 较分散，宽基ETF常见
< 30%  → 高度分散，有效分散个股风险

单一持仓 > 15% → ⚠️ 单股集中度过高（须注明）
```

**4.2 行业/板块分布**（股票型ETF必须）

列出持仓前3行业及各自占比，判断：
- 行业集中型（单一行业 > 60%）：行业ETF属于正常，但需评估行业景气度
- 跨行业分散型（无单一行业 > 30%）：宽基ETF特征，系统性风险更低

**4.3 持仓质量评估**（行业/主题ETF重点关注）

```
□ 是否存在流动性极差的成分股（影响TE）
□ 是否包含ST/退市风险股（需⚠️）
□ 成分股集中于大市值龙头 vs 中小市值（波动风险不同）
□ 指数编制规则是否合理（市值加权 vs 等权 vs 因子加权）
```

**4.4 债券ETF持仓补充**（仅债券型）

```
□ 久期（Duration）：X年  → 利率敏感度
□ 信用评级分布：AAA占X% / AA+占X% / AA及以下占X%
□ 到期收益率（YTM）：X%
□ 可转债ETF：正股溢价率均值 / 纯债价值 / 转股期分布
```

**4.5 商品ETF持仓补充**（仅商品型）

```
□ 实物持有 vs 期货合约（展期成本⚠️）
□ 黄金ETF：实物金托管机构 / 托管费 / 实物兑换条款
□ 期货ETF：近月/次近月合约展期规则 / 历史年化展期成本
```

## M5｜指数估值分位分析（择时核心，25%）

> 对于股票型ETF，这是最重要的模块，决定"现在该不该买"。

### 股票型ETF（宽基/行业/主题）

**5.1 PE分位估值**

```
PE历史分位 = (当前PE - N年最低PE) / (N年最高PE - N年最低PE) × 100%
（另一种算法：当前PE低于历史N年中X%的时间点）

参考分位区间（适用多数宽基指数）：
< 20%  → 历史低估区间，逆向布局窗口
20~40% → 相对低估，正常配置
40~60% → 历史均值附近，合理区间
60~80% → 偏贵，谨慎加仓
> 80%  → 历史高估区间，控制仓位

⚠️ 行业ETF的PE分位阈值因行业不同差异显著，须与行业历史基准比较，不得套用宽基标准。
⚠️ 科技/成长型指数PE天然高于价值型指数，不可跨指数类型横向比较绝对值。
```

**5.2 PB分位估值**

```
PB分位方法同PE，对银行/地产等重资产行业更有参考意义。
PB < 1.0 → 破净，历史罕见，须确认是否为价值陷阱（行业衰退/资产减值）
```

**5.3 隐含回报率**（盈利收益率法，格雷厄姆适用于成熟指数）

```
盈利收益率 = 1 / PE（即 E/P）
当盈利收益率 > 10年期国债收益率 × 1.5 → 股票相对债券有吸引力
```

**5.4 多时间维度验证**

| 时间维度 | 3年分位 | 5年分位 | 10年分位（若有）| 综合判断 |
|---------|--------|--------|--------------|---------|

## M6｜资金流向 & 规模趋势分析（10%）

> 资金行为是市场情绪的映射，辅助判断市场共识方向。

**6.1 规模趋势**

```
AUM近6个月变化趋势：
持续增长（>10%/季）→ 市场看多，资金净流入
基本稳定（±10%/季）→ 中性
持续缩减（>10%/季）→ ⚠️ 资金净流出，需关注原因（市场下跌 vs 主动赎回）

注意区分：规模缩减 = 净值下跌（被动缩减）or 净赎回（主动流出）
净赎回比净值下跌更值得警惕，说明投资者在主动离场。
```

**6.2 换手率趋势**

```
换手率突然放大（>日均2倍）→ 短期情绪波动，谨慎追涨
换手率持续萎缩 → 关注度下降，流动性可能进一步恶化
行业ETF换手率暴涨 → 往往是概念炒作信号，非价值信号 ⚠️
```

**6.3 A股特有情绪信号**

```
【谨慎追涨情境】
- ETF近30日涨幅超过30%（情绪可能已price-in）
- 社交媒体/财经媒体大规模报道该行业主题
- 行业ETF规模短期翻倍（追热钱涌入）

【可能的逆向信号（需配合估值使用）】
- 规模持续缩减但PE分位处于历史低位 → 悲观情绪可能过度
- 换手率萎缩但基本面未恶化 → 市场冷淡期
```
