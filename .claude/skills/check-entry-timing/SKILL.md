# check-entry-timing

**这个 skill 只回答一件事：**
> 如果我想配置这个指数，现在是不是好的介入时点？

它不重复做估值判断（那是 choose-index-now 的工作），而是专注于：**趋势方向够不够稳、动量信号有没有确认、情绪有没有透支。**

---

## 使用方式

```
/check-entry-timing [指数名称或代码]

示例：
/check-entry-timing 创业板指（399006）
/check-entry-timing 纳斯达克100
/check-entry-timing 沪深300（000300）
```

## 信号一：三因子动量评分

> 判断当前趋势方向和质量，区分"流畅上涨"与"震荡蹉跎"。
> 来源：猫哥AI量化多因子动量策略（乖离+斜率+效率三因子融合）。

**因子一：斜率动量（权重 30%）**

```
对近60日收盘价做归一化后线性回归：
normalized = close / close[0]
斜率动量得分 = slope × R² × 10000

slope > 0 且 R² > 0.6 → 趋势强且稳定 🟢
slope > 0 且 R² < 0.3 → 走势杂乱，趋势不可信 🟡
slope < 0             → 趋势向下 🔴
```

**因子二：乖离动量（权重 30%）**

```
bias = 收盘价 / 近90日均线
对最近25日乖离序列做线性回归：
乖离动量得分 = 乖离斜率 × 10000

得分 > 0 → 价格偏离均线趋势在扩大（强势阶段）🟢
得分 < 0 → 价格回归均线或反向（弱势阶段）🔴
```

**因子三：效率动量（权重 40%）**

```
pivot = (开盘 + 最高 + 最低 + 收盘) / 4
momentum = 100 × ln(pivot[-1] / pivot[0])
efficiency = 净移动距离 / 总波动距离
效率动量得分 = momentum × efficiency

得分高 → 涨幅大且走势流畅，趋势质量高 🟢
得分低 → 有涨幅但震荡频繁，不稳定 🟡
```

**三因子合成（Z-Score 标准化后加权）**

```
composite = 0.30×z_斜率 + 0.30×z_乖离 + 0.40×z_效率

> 1.0   → 🟢🟢 动量强劲，顺势方向明确
0 ~ 1.0 → 🟢   动量温和，谨慎参与
-1 ~ 0  → 🟡   动量偏弱，降低仓位敞口
< -1.0  → 🔴   动量崩溃，不宜持有
```

```

composite：[X]  → [🟢🟢强劲 / 🟢温和 / 🟡偏弱 / 🔴崩溃]
```

## 信号二：RSRS + 均线双重确认

> RSRS 量化"上涨空间 vs 下跌空间"的相对强度，均线确认短期趋势方向。
> 来源：光大证券2017研报《基于阻力支撑相对强度的市场择时》。

```
RSRS 计算：
① 近18日以低价为X、高价为Y做线性回归，得斜率 β 和 R²
② 维护近600日 β 序列，计算当日 Z-Score
③ RSRS得分 = Z-Score × R²

信号阈值：
> 0.7    → 🟢 强势买入信号
-0.7~0.7 → 🟡 震荡，维持现状
< -0.7   → 🔴 弱势清仓信号
```

```
RSRS + 均线双重确认：

RSRS > 0.7  且 MA20今日 > MA20五日前 → ✅ 双重买入确认（高置信）
RSRS < -0.7 且 MA20今日 < MA20五日前 → ❌ 双重清仓确认（高置信）
两者方向冲突                          → ⚠️ 降低置信度，继续等待
```

```
【RSRS 信号】
RSRS得分：[X]       → [🟢买入 / 🟡震荡 / 🔴清仓]
MA20 方向：[上/横/下]
双重确认：[✅买入确认 / ⚠️信号冲突 / ❌清仓确认]
```

## 信号三：情绪拥挤度（可选，获取困难时跳过）

> 情绪数据不稳定、依赖外部数据源，作为辅助参考而非决策依据。
> 若数据无法获取，标注"[数据缺失]"并跳过本节，不影响主结论。

```
搜索热度（近30日 vs 近90日均值）：
> 200%  → 🔴 情绪过热
80~200% → 🟡 正常
< 80%   → 🟢 市场冷清（逆向关注）

社媒情绪（看多比例）：
> 80%   → 🔴 乐观共识过强
20~80%  → 🟡 正常
< 20%   → 🟢 极度悲观（逆向信号）

情绪综合温度：
🌡️ 热（两项均热）  → ⚠️ 与估值交叉验证，高估值时减仓预警
🌡️ 中            → 正常持有
🌡️ 冷（两项均冷）  → 🟢 可能是逆向布局窗口
```

```
【情绪拥挤度】（可选）
搜索热度：[X% / 数据缺失]
社媒看多比：[X% / 数据缺失]
情绪温度：[🌡️ 热/中/冷 / 数据缺失，跳过]
```
